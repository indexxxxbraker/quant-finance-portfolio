# Sobol direction numbers (Joe-Kuo)

This directory contains the Sobol sequence direction-numbers file used
by the C++ QMC implementation in `cpp/src/qmc.cpp`.

## File

- `new-joe-kuo-6.21201`: direction numbers for Sobol sequences in up
  to 21,201 dimensions, obtained by Joe and Kuo (2008) using their
  search criterion D(6).

The file is consumed at runtime by `quant::Sobol::Sobol(...)` via the
path baked into the binary by `CMakeLists.txt` (the
`SOBOL_DATA_FILE` macro).

## Source and citation

This file was downloaded from the Joe-Kuo Sobol sequence generator
website maintained by Frances Kuo at the University of New South
Wales:

  https://web.maths.unsw.edu.au/~fkuo/sobol/

If you use these direction numbers in published work, please cite:

  S. Joe and F. Y. Kuo, "Constructing Sobol sequences with better
  two-dimensional projections", SIAM Journal on Scientific Computing,
  30(5):2635-2654, 2008.

## File format

Each non-header line corresponds to one dimension and has the form:

```
d  s  a  m_1  m_2  ...  m_s
```

where:

- `d`: dimension number (starts at 2 since dimension 1 is trivial).
- `s`: degree of the primitive polynomial used for this dimension.
- `a`: integer encoding of the polynomial coefficients
  $a_1, \ldots, a_{s-1}$ (each 0 or 1, packed into the bits of `a`,
  with $a_1$ as the most-significant bit).
- `m_1, ..., m_s`: initial direction numbers (positive integers).

The direction numbers $m_k$ for $k > s$ are generated by the
recurrence given in Joe & Kuo (2008), implemented in
`Sobol::load_direction_numbers` in `qmc.cpp`.
