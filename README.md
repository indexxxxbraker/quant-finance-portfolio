# Quantitative Finance Portfolio

A collection of derivative pricing algorithms implemented in Python and C++, accompanied by rigorous mathematical write-ups.

## Author

Roberto Cepeda Rocaná — Mathematics & Physics undergraduate, aspiring Quantitative Analyst.

## Project structure

- **`docs/`** — LaTeX write-ups with the mathematical theory behind each implementation.
- **`python/`** — Reference implementations in Python.
- **`cpp/`** — Production-style implementations in C++.
- **`notebooks/`** — Demonstration notebooks.
- **`tests/`** — Unit tests.

## Roadmap

- [ ] **Phase 1 — European options.** Black-Scholes analytical formula, Greeks, implied volatility.
- [ ] **Phase 2 — Monte Carlo.** GBM simulation, variance reduction, pathwise/likelihood-ratio Greeks.
- [ ] **Phase 3 — Trees and PDEs.** Binomial/trinomial trees, finite-difference schemes, American options.
- [ ] **Phase 4 — Exotic options and advanced models.** Barrier, Asian, lookback options; local/stochastic volatility.

## References

- Glasserman, P. (2004). *Monte Carlo Methods in Financial Engineering.*
